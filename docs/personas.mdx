---
title: Personas
description: "Generate and read About-Me personas scoped to a namespace."
---

The persona endpoints expose a four-call lifecycle on top of the
`/gateway/v1/personas` surface: **create** a binding, **build** the persona
asynchronously, **poll** its status, and **read** the composite document.

The SDK hides the underlying `agent_id` behind namespace-scoped resolution —
every method accepts either `namespace_id` or `external_reference_id`.

## Create

```ts
const binding = await deyta.personas.create({
  namespace_id: "ns_123",
  subject: "Alice",
});
// binding: { agent_id, namespace_id, role: "primary", subject, ... }
```

The first call creates the primary binding (HTTP 201); subsequent calls return
the existing binding (HTTP 200). `subject` is honored on the first call only —
later calls with a different subject silently keep the original.

## Build

```ts
const { build_id } = await deyta.personas.build({ namespace_id: "ns_123" });
```

Fire-and-forget. Returns HTTP 202 with a `build_id` for log correlation. Not
idempotent — the gateway returns `409 CONFLICT` if a build is already in flight.
Poll `status()` to follow progress.

## Status

```ts
const { status, last_built_at } = await deyta.personas.status({
  namespace_id: "ns_123",
});
// status: "building" | "ready" | "not_built"
```

## Read

```ts
const persona = await deyta.personas.read({ namespace_id: "ns_123" });
// persona: { agent_id, identity, traits, episodes, peers, facets, providers, ... }
```

Returns `404 NOT_FOUND` when no binding exists **or** when the binding has not
been built yet — call `status()` to disambiguate.

## Scoped form

<CodeGroup>
```ts Scoped (recommended)
const ns = deyta.namespaces.scope("ns_123");

await ns.personas.create({ subject: "Alice" });
await ns.personas.build();
await ns.personas.status();
await ns.personas.read();
```

```ts Top-level
await deyta.personas.create({ namespace_id: "ns_123", subject: "Alice" });
await deyta.personas.build({ namespace_id: "ns_123" });
await deyta.personas.status({ namespace_id: "ns_123" });
await deyta.personas.read({ namespace_id: "ns_123" });
```
</CodeGroup>

<Note>
  Provide exactly one of `namespace_id` or `external_reference_id` per call.
  Passing both, or neither, is a client error.
</Note>
